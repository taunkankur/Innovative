• SIMS - School Information Management System

  • Abstract

  • This was an attempt to enhance my previous project SCAM that I did at my college. Along with attendance information of students, we aimed at developing a product that would have eased school's operation. We upgraded the tool to record information of students e.g academic record - marks, achievement, awards(if any); official record - fees related; customized SMS notification to parents for some events at college etc.

  • Technical Specification

  • 1. Operating System - Windows
  • 2. Data Base - SQL
  • 3. Language - Java/JavaScript/HTML/CSS

  • SCAM - School Children Attendance Monitoring

  • Abstract

  • Speaking technically, it involved RFID, GSM Module, Micro controller, LCD, Windows server, Java, Communication protocol, DB. Through RFID, attendance of school children were captured automatically and simultaneously respective parents were updated though SMS. Also, data base was updated regarding the attendance. This tool also had feature to screen report relating to attendance of particular student. It was a wholesome experience to integrate so many components, though their were many impediments, which were overcome with the help of able professors at college.

  • Technical Specification

  • 1. Operating System - Windows
  • 2. Data Base - SQL
  • 3. Components - GSM Module/RFID/8051 Micro-controller/LCD
  • 2. Language - Java

  • Automation in Irrigation

  • Abstract

  • Speaking technically, we had integrated LCD, Keyboard and electronic valves with micro controller. The main purpose was to sprinkle water to crops in controlled manner as and when required by the user. The data was fetched using keyboard and displayed in LCD, depending upon which, the valve operated.

  • Technical Specification

  • 1. Components - DTMF Keyboard/8051 Micro-controller/LCD
  • 2. Language - 8051 Micro-controller programming
